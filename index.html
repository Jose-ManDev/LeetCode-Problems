1. ¿Cuál es la diferencia entre constante y variable?

    Constantes: Las constantes son lugares de memoria que no pueden ser cambiados una vez han sido definidosy pueden ser de todos los tipos.

    Variables: Las variables, al igual que las constantes son lugares de memoria, pero a diferencia de estas pueden ser reasignadas en cualquier momento a un valor del mismo tipo que fueron declaradas.

  2. En el lenguaje C, ¿cómo se definen las constantes y las variables?

    Las constantes se definen de la siguiente manera:

    #include <stdio.h>

    // Declarar una constante usando define
    #define constant1 10
    int main() {
      // Declarar una constante usando const
      const int constant2 = 10;
    }

    Las variables se definen de la siguiente manera:

    #include <stdio.h>

    int main() {
      // Declaración de una variable
      int var1;
      // Asignación de la variable
      var1 = 10;

      // Declaración y asignación de una variable
      float var2 = 2.5;

      // Declarar varias variables
      char var3, var4, var5;
    }

  3. Independientemente del lenguaje de programación, ¿cuáles son los tipos de datos que existen?

    Estos son los tipos de datos que existen:

    • Entero o int
    • Flotante o float
    • Doble o double
    • Carácter o char
    • Booleano o boolean
    • Vacío o void

    Algunos lenguajes también manejan el tipo string o cadena o los números complejos

  4. En lenguaje C, ¿cuáles son los tipos de datos simples?

    En lenguaje C, estos son los datos simples o primitivos:

    Tipo Descripción Ejemplo
    Entero El tipo entero se usa para representar números enteros (es decir, sin parte decima) y se usa mediante la palabra clave int
    int age = 30;
    Carácter El tipo carácter se usa para representar caracteres ASCII, estos se pueden usar mediante la palabra clave char
    char initialLetter = 'J';
    Punto flotante Es usado para representar números reales, es decir, aquellos con parte decimal. Se usa mediante la palabra clave float o double, el cual ocupa el doble de memoria pero da más precisión
    float price = 24.55;
    Vacío Si bien este no es un tipo como tal, es usado para indicar que una función no regresa ningún valor mediante la palabra clave void
    void myFunction() { ... }
    Lógico o booleano Este no es un tipo nativo de C y para usarlo se debe de agregar importar una biblioteca usando #include <stdbool.h> . Se puede usar mediante la palabra clave bool
    bool isArrayEmpty = true;

  5. Describe cinco bibliotecas del lenguaje C

    Biblioteca: stdio.h
    Descripción Esta biblioteca proporciona una amplia gama de funciones de entrada/salida en C, ya sea en pantalla o en archivos.
    Función printf
    Objetivo Esta función toma una cadena de formato y argumentos opcionales para generar una cadena con formato para la salida.
    Función scanf
    Objetivo Esta función se usa para obtener una entrada del usuario a través del teclado y la almacena en la variable especificada.
    Función fopen
    Objetivo Esta función abre el archivo de la ruta dada. No es una función segura por lo que no se realiza una validación al archivo antes de abrirlo.
    Biblioteca: stdlib.h
    Descripción Esta biblioteca proporciona una amplia gama de funciones para la gestión de memoria, el control de procesos y otros usos.
    Función rand
    Objetivo Esta función genera un número entero pseudo-aleatorio, no es criptográficamente segura.
    Función malloc
    Objetivo Esta función reserva un espacio de memoria del número de bytes que se le pide.
    Función abort
    Objetivo Esta función termina la ejecución del programa inmediatamente, no acepta parámetros.
    Biblioteca: string.h
    Descripción Esta biblioteca proporciona una amplia gama de funciones para la manipulación de cadenas de caracteres, además de la definición de macros, constantes, tipos y algunas funciones para manipular memoria.
    Función strcat
    Objetivo Esta función concatena dos cadenas de texto dadas.
    Función memcpy
    Objetivo Esta función copia n bytes de un buffer de memoria a otro, los buffer no pueden solaparse.
    Función strlen
    Objetivo Esta función devuelve la longitud de una cadena de texto, es decir, cuantos caracteres tiene.
    Biblioteca: math.h
    Descripción Esta biblioteca proporciona una amplia gama de funciones para operaciones matemáticas básicas, muchas de sus funciones incluyen el uso de números con coma flotante.
    Función cos
    Objetivo Esta función calcula el coseno del valor dado en radianes.
    Función floor
    Objetivo Esta función regresa el número entero más cercano sin pasarse al valor dado. Si le pasamos 2.8 esta regresará el valor 2.0000.
    Función pow
    Objetivo Esta función eleve un valor dado a la potencia dada.
    Biblioteca: time.h
    Descripción Esta biblioteca proporciona una amplia gama de funciones que sirven para manipular y dar formato a la hora y fecha del sistema.
    Función clock
    Objetivo Esta función cuenta el número de pulsos desde que se inició un proceso y los devuelve.
    Función difftime
    Objetivo Esta función devuelve la diferencia entre dos variables de tiempo en segundos.
    Función time
    Objetivo Esta función devuelve la hora y fecha actual o -1 si no es posible.

  6. En C, ¿por qué es importante utilizar el tipo de variable correcto y por qué es importante utilizar bibliotecas?

    A la hora de trabajar con datos en C es importante usar el tipo de variable correcto porque el tipo de variable determina el rango y la precisión de los valores que se almacenan en ella, así como las operaciones que se pueden realizar en ellos.

    Al usar el tipo de variable equivocado se pueden provocar errores que incluyen resultados poco precisos o incluso incorectos. Además, el uso de los tipos de variable incorrectos puede consumir más memoria de la necesaria, lo que puede ralentizar el rendimiento del programa.

    Uno de los problemas más comunes al escoger un tipo de variable incorrecto es el llamado desbordamiento. "El desbordamiento es la condición que se da cuando una operación aritmética, excede el tamaño máximo del tipo de variable usado para almacenarlo. Cuando el desbordamiento sucede, el valor interpretado parecerá como si le hubiese dado la vuelta y empezado desde el valor mínimo" (Dietz et al., 2015).

    Igualmente, a la hora de programar las librerías nos dan una serie de ventajas que no debemos de olvidar. Una de sus principales ventajas es la reutilización de código, ya que en vez de programar nuestra propia implementación podemos usar una función que ya existe. Esto nos ahorra tiempo y recursos a la hora de desarrollar software, además, normalmente las funciones dentro de una librería superarán lo que sea que uno programe.
